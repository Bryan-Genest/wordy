# Wurd 

Kinda like Wordle but not really

Ruby version 2.7.3

## Installation
* Run ```yarn install```
* Run ```bundle install```

## Starting the server
* Run ```yarn start``` to start the server
* Run ```rails s``` to start the React front-end

## Database creation
* ```bundle exec rake db:create```
* ```bundle exec rake db:migrate```
* ```bundle exec rake db:seed```
